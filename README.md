Projekti kirjeldus:

Tegemist on Chuck Norrise naljade lehitsemise lehega. Lehel saab kasutada 'random' nalja leidjat ning on olemas lehekülg, kust saab kuvada API'ist tõmmatud naljade loetelu.
Loetelust saab otsida nalju kasutades otsingu riba, valida erinevaid API'i naljade kategooriaid ning määrata lemmik nalju.
Naljadel vajutades näeb rohkem API'ist pärenevat nalja meta andmeid.

API dokumentatsioon:
https://api.chucknorris.io/

Rakenduse käivitamiseks on vajalik installeerida vajalikud NPM moodulid. Selleks.
 1. kasuta CMD'd käsklus cd ja liigu API_CALL_BASED_SITE kausta
 2. kirjuta CMD's käsklus NPM update (võib ka NPM install).
 Nüüd peaks toimuma vajalike NPM moodulite install.

 Käivitamiseks kohalikus arvutis kastua CMD's käsklus npm run dev. Terminal annab kohaliku arvuti aadressi, kus lehekülg on käivitatud.
